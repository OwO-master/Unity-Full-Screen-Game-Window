1. PANIK! (Shift+F12)  
   >Set Game window to (main screen):  
      >position (x,y)=(0,0)  
      >size (w,h)=(800,600)  
      
2. Refresh (Ctrl+F12)  
   >Try to find any detached Game window   
  
3. FullScreen (F12)  
   >Toggle Game window to  
   >  1.FullScreen  
   >    position (main screen) (x,y)=(-7,-42)  
   >    size (w,h)=(1934,1129)  
   >  2.Pre-Fullscreen position and size
